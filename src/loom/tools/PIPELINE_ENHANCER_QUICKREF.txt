1. research_enhance()
   ─────────────────────────────────────────────────────────────────────────
   Execute ANY tool with automatic enrichment

   Signature:
     async def research_enhance(
         tool_name: str,
         params: dict[str, Any],
         auto_hcs: bool = True,
         auto_cost: bool = True,
         auto_learn: bool = True,
         auto_fact_check: bool = False,
         auto_suggest: bool = True,
     ) -> dict[str, Any]

   Returns:
     {
       "_original_result": {...},       # Tool output
       "_hcs_scores": {...},            # 8-dimension quality scores
       "_estimated_cost": {...},        # Pre-execution cost estimate
       "_actual_cost": {...},           # Post-execution actual cost
       "_suggested_tools": [...],       # Related tools for follow-up
       "_fact_check": {...},            # Verified factual claims
       "_learning_recorded": bool,      # Strategy learning status
       "_execution_time_ms": int,       # Wall-clock execution time
       "_error": str,                   # Error message (if any)
     }

   Example:
     result = await research_enhance(
         tool_name="research_deep",
         params={"query": "AI safety vulnerabilities"},
         auto_hcs=True,
         auto_cost=True,
         auto_learn=False,
         auto_fact_check=False,
         auto_suggest=True,
     )


2. research_enhance_batch()
   ─────────────────────────────────────────────────────────────────────────
   Execute multiple tools in parallel with enhancement

   Signature:
     async def research_enhance_batch(
         tasks: list[dict[str, Any]],
     ) -> dict[str, Any]

   Task dict format:
     {
       "tool_name": str,                # Required: tool name
       "params": dict,                  # Required: tool parameters
       "auto_hcs": bool,                # Optional: default True
       "auto_cost": bool,               # Optional: default True
       "auto_learn": bool,              # Optional: default True
       "auto_fact_check": bool,         # Optional: default False
       "auto_suggest": bool,            # Optional: default True
     }

   Returns:
     {
       "results": [...],                # List of individual results
       "total_time_ms": int,            # Total batch execution time
       "success_count": int,            # Number of successful executions
       "error_count": int,              # Number of failures
     }

   Example:
     result = await research_enhance_batch([
         {
             "tool_name": "research_fetch",
             "params": {"url": "https://example.com"},
             "auto_hcs": True,
         },
         {
             "tool_name": "research_search",
             "params": {"query": "AI regulations"},
             "auto_suggest": True,
         },
     ])

================================================================================
                          ENRICHMENT LAYERS
================================================================================

PRE-EXECUTION:
  1. Cost Estimation (auto_cost=True)
     - Estimates tokens, API calls, cost before execution
     - Non-blocking, zero overhead

POST-EXECUTION (parallel):
  2. HCS Scoring (auto_hcs=True)
     - 8-dimension quality assessment
     - Dimensions: bypass_success, information_density, stealth_score,
       transferability, persistence, escalation_potential, defense_evasion,
       novelty
     - Latency: +50-200ms

  3. Strategy Learning (auto_learn=True)
     - Records to meta-learner if params contain reframe data
     - Triggered by: strategy_name, prompt_reframed, reframe_strategy
     - Latency: +20-50ms

  4. Fact Checking (auto_fact_check=True)
     - Verifies factual claims in output
     - SLOW: +1-3 seconds per result
     - Default: False (disabled)

  5. Tool Suggestions (auto_suggest=True)
     - Recommends follow-up tools based on output
     - Similarity-based ranking
     - Latency: +50-150ms

================================================================================
                          PARALLEL EXECUTION
================================================================================

All post-execution enrichments run concurrently:

  async def research_enhance():
      result = await _execute_tool()  # Sequential

      tasks = [
          _score_with_hcs(...),       # Concurrent
          _feed_to_meta_learner(...), # Concurrent
          _verify_factual_claims(...),# Concurrent
          _suggest_follow_up_tools(...),# Concurrent
      ]

      results = await asyncio.gather(*tasks, return_exceptions=True)

Total latency ≈ max(task_time) not sum(task_times)

================================================================================
                          ERROR HANDLING
================================================================================

All enrichment is NON-BLOCKING:
  - Individual enrichment failures logged but don't crash pipeline
  - Tool execution errors recorded in "_error" field
  - Results still valid even if enrichment partially fails

Error Field:
  "_error": "Tool execution failed: ..."

Graceful Degradation:
  - HCS scoring fails? Log and use default scores
  - Fact-checking fails? Skip and continue
  - Tool suggestion fails? Still return other enrichments
