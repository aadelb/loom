USAGE QUICK START
=================

Basic usage:
  import asyncio
  from loom_sdk import LoomClient

  async def main():
      async with LoomClient() as client:
          results = await client.search("query", n=10)
          print(results)

  asyncio.run(main())

With authentication:
  client = LoomClient(
      base_url="https://api.example.com",
      api_key="sk_live_..."
  )

See README.md for complete documentation and examples.

API ENDPOINTS
=============

GET /health
  HealthStatus check

POST /api/search
  Multi-provider semantic search
  Query parameters: query, provider, n

POST /api/fetch
  Extract content from URL
  Query parameters: url, mode

POST /api/deep-research
  12-stage research pipeline
  Query parameters: query, max_results

POST /api/llm/summarize
  LLM-powered text summarization
  Query parameters: text, max_words, model

POST /api/tools/{tool_name}
  Generic tool invocation
  Query parameters: tool-specific
