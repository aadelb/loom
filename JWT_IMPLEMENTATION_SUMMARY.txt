QUICK START:
============

1. Install PyJWT:
   pip install PyJWT

2. Set environment variable:
   export LOOM_JWT_SECRET="your-secure-key"

3. Create a token:
   from loom.jwt_auth import create_token
   token = create_token("user@example.com", "researcher", expires_in_hours=168)

4. Validate token:
   from loom.jwt_auth import validate_token
   payload = validate_token(token)

5. Check tool access:
   from loom.jwt_auth import check_tool_access
   if check_tool_access(token, "research_fetch"):
       # Proceed with tool execution
