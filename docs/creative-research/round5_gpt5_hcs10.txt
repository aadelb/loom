## 1. Grant Language Forensics
**Concept.**  Mine linguistic fingerprints of scientific funding proposals by combining NSF award abstracts, NIH RePORTER data, OpenAIRE grants, OpenAlex abstracts, and DBLP titles. Zipf exponents and Benford‐law digit frequencies, when compared across disciplinary clusters, reveal linguistic drift and budget inflation biases invisible to any single corpus.

**Mathematical base.**  Zipf’s law for term‑frequency rank relations; Benford’s law goodness‑of‑fit χ²; cosine‑distance of topic embeddings.

**APIs.**  
1. https://api.nsf.gov/services/v1/awards.json  
2. https://api.reporter.nih.gov/v2/projects/search  
3. https://api.openaire.eu/search/projects  
4. https://api.openalex.org/works  
5. https://dblp.org/search/publ/api  

```python
import asyncio, aiohttp, math, collections
from typing import TypedDict, List, Dict
class GrantStats(TypedDict):
    source:str; zipf_exponent:float; benford_chi2:float

async def fetch_json(session,url,params=None):
    async with session.get(url,params=params) as r:return await r.json()

def zipf_exponent(freqs:Dict[str,int])->float:
    ranks=sorted(freqs.values(),reverse=True)
    logf=[math.log(f) for f in ranks if f>0]
    logr=[math.log(i+1) for i,_ in enumerate(ranks)]
    n=len(logf)
    sxy=sum(a*b for a,b in zip(logr,logf))
    sx,sy=sum(logr),sum(logf)
    sxx,sumy=n*sum(a*a for a in logr),n*sum(logr)**2
    return -(n*sxy-sx*sy)/(sxx-sumy)

def benford_chi2(nums:List[float])->float:
    import numpy as np
    digs=np.array([int(str(abs(x))[0]) for x in nums if x!=0])
    obs=np.bincount(digs, minlength=10)[1:]
    exp=np.array([math.log10(1+1/d) for d in range(1,10)])*len(digs)
    return float(((obs-exp)**2/exp).sum())

async def main():
    urls=["https://api.nsf.gov/services/v1/awards.json","https://api.openalex.org/works"]
    async with aiohttp.ClientSession() as s:
        tasks=[fetch_json(s,u) for u in urls]
        data=await asyncio.gather(*tasks)
    out=[]
    for src,j in zip(urls,data):
        text=" ".join(str(j).split())
        words=text.lower().split()
        freqs=collections.Counter(words)
        nums=[float(x) for x in range(1,100)]
        out.append(GrantStats(source=src,
                              zipf_exponent=zipf_exponent(freqs),
                              benford_chi2=benford_chi2(nums)))
    return out

if __name__=="__main__":print(asyncio.run(main()))
```

## 2. Methodological Monoculture Index
**Concept.**  Quantifies how diverse methodological vocabularies are across papers’ methods sections. By merging PubMed Central full‑text snippets, arXiv metadata, CrossRef abstracts, bioRxiv API, and Semantic Scholar, it converts technique terms into feature distributions; Shannon entropy and Simpson diversity compute cross‑field methodological richness.

**Statistical base.**  Shannon entropy H = –∑ p log p; Gini‑Simpson 1 – ∑ p²; Jensen–Shannon divergence across corpora.

**APIs.**
1. https://api.ncbi.nlm.nih.gov/lit/ctxp/  
2. https://api.crossref.org/works  
3. https://api.semanticscholar.org/graph/v1/paper/search  
4. https://api.biorxiv.org/details/  
5. https://api.arxiv.org/query  

```python
import asyncio, aiohttp, math, collections
from typing import TypedDict, Dict
class Diversity(TypedDict):
    source:str; shannon:float; simpson:float

async def fetch(session,url,params=None):
    async with session.get(url,params=params) as r:return await r.json()

def diversity(stats:Dict[str,int])->Dict[str,float]:
    n=sum(stats.values()); p=[v/n for v in stats.values() if v>0]
    H=-sum(pi*math.log(pi) for pi in p)
    S=1-sum(pi*pi for pi in p)
    return {"shannon":H,"simpson":S}

async def main():
    urls=["https://api.crossref.org/works","https://api.semanticscholar.org/graph/v1/paper/search"]
    async with aiohttp.ClientSession() as s: data=await asyncio.gather(*[fetch(s,u) for u in urls])
    out=[]
    for src,j in zip(urls,data):
        words=str(j).lower().split(); freq=collections.Counter(words)
        d=diversity(freq); out.append(Diversity(source=src,**d))
    return out

if __name__=="__main__":import pprint; pprint.pp(asyncio.run(main()))
```
