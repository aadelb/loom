b. Immediate intent to return – evidenced by travel to UAE, e-mail trail, bringing phone to airport in same sealed state (ask for forensic “no user activity” report).  
c. Unlawful identification: CCTV only shows man picking up “an object”; serial number link missing – raise chain-of-custody doubt.

f. Deportation discretionary: request court to exercise Art 126/2 phrase “يجوز للمحكمة” (may order) – present employer letter, stable residency.
