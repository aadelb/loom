## Tool 1: research_persona_profile
Location: /Users/aadel/projects/loom/src/loom/tools/persona_profile.py
Lines: 389
Type: Async MCP tool

Features:
- Cross-platform persona reconstruction from text samples
- Linguistic analysis: formality level (0-1), vocabulary tier (basic/intermediate/advanced)
- Big Five personality indicators:
  * Openness: diverse vocabulary, abstract thinking
  * Conscientiousness: structured sentences, precise language
  * Extraversion: social words, exclamation marks, informal tone
  * Agreeableness: positive sentiment, inclusive pronouns, hedging
  * Neuroticism: negative emotion words, anxiety markers
- Temporal pattern analysis: peak hours, timezone estimate, activity patterns
- Topic extraction: top 10 distinctive nouns/noun phrases
- Education level estimation: high_school, undergraduate, graduate, unknown
- Optional LLM assessment for personality overview

Return signature:
{
  "profile": {
    "formality": float,  # 0-1
    "vocabulary_tier": str,  # basic|intermediate|advanced
    "personality": {
      "openness": float,
      "conscientiousness": float,
      "extraversion": float,
      "agreeableness": float,
      "neuroticism": float,
    },
    "top_topics": [str],
    "estimated_education": str,
  },
  "temporal": {  # optional, if metadata with timestamps provided
    "peak_hours": [int],
    "timezone_estimate": str,
    "activity_pattern": str,  # nocturnal|diurnal|irregular
  } | None,
  "text_count": int,
  "total_words": int,
  "llm_assessment": str | None,
}

## Tool 2: research_radicalization_detect
Location: /Users/aadel/projects/loom/src/loom/tools/radicalization_detect.py
Lines: 328
Type: Async MCP tool

Features:
- Radicalization indicator detection using NLP heuristics
- Six indicator categories with individual scoring (0-1):
  1. Us-vs-them framing: pronoun ratio analysis + group identity markers
  2. Extremist vocabulary: keywords like jihad, crusade, martyr, liberation, etc.
  3. Escalation language: destroy, eliminate, kill, bomb, attack, etc.
  4. Moral absolutism: binary language (always/never), absolute judgments
  5. Dehumanization: animal/disease metaphors (vermin, plague, parasite, etc.)
  6. Call to action: imperative verbs + urgency words (must/now/immediately)

- Overall risk score: weighted average (0-1)
- Risk level classification: low (<0.2), moderate (0.2-0.4), elevated (0.4-0.7), high (>0.7)
- Optional context parameter for background information
- Optional LLM assessment for nuanced analysis

Return signature:
{
  "risk_score": float,  # 0-1
  "risk_level": str,  # low|moderate|elevated|high
  "indicators": {
    "us_vs_them": {"score": float, "examples": [str]},
    "extremist_vocabulary": {"score": float, "terms_found": [str]},
    "escalation_language": {"score": float, "terms_found": [str]},
    "moral_absolutism": {"score": float, "terms_found": [str]},
    "dehumanization": {"score": float, "terms_found": [str]},
    "call_to_action": {"score": float, "examples": [str]},
  },
  "llm_assessment": str | None,
  "word_count": int,
  "context": str | None,
}
