IMPLEMENTED TOOLS:
===================

1. research_prompt_injection_test(target_url, model_name, test_count=10)
   - Tests for prompt injection vulnerabilities
   - 20 hardcoded test vectors covering:
     * System prompt extraction
     * Instruction override attempts
     * Role confusion tactics
     * Context injection techniques
   - Returns: bypasses_detected count + detailed results

2. research_model_fingerprint(target_url, probes=5)
   - Identifies LLM model behind API endpoint
   - 5 capability probes testing:
     * Math arithmetic capability
     * Code generation ability
     * Language knowledge
     * Knowledge cutoff date
     * Logical reasoning
   - Returns: likely_model name, confidence score 0-1, probe results

3. research_bias_probe(target_url, categories=None)
   - Tests for demographic bias in responses
   - Default 4 categories: gender, ethnicity, age, disability
   - Template-based prompts with different demographic groups
   - Returns: bias_detected flag, per-category bias scores 0-1

4. research_safety_filter_map(target_url, categories=None)
   - Maps safety filter boundaries via binary search
   - 4 categories: violence, substances, weapons, personal_info
   - Graduated prompts from benign to sensitive (5 levels each)
   - Returns: threshold levels and first-blocked/last-allowed boundaries

5. research_compliance_check(domain, frameworks=None)
   - Checks website for AI compliance indicators
   - 3 frameworks: eu_ai_act, gdpr, ccpa
   - Fetches privacy policy, terms, cookie consent pages
   - Searches for compliance keywords
   - Returns: compliance_score 0-100, findings per framework/requirement

HELPER FUNCTIONS:

- _infer_model_from_indicators(): Model inference from capability indicators
- _calculate_bias_score(): Bias score based on response length variance
