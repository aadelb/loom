2.  **CLI Discoverability:** There is no way for a user to discover the 94 tools from the terminal. The CLI acts as a hardcoded wrapper for a small subset rather than a dynamic MCP client.
3.  **Documentation Gap:** Between 41 and 54 tools are completely undocumented across both markdown files.
