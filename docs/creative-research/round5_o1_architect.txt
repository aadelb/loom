2) Tool Name & Description: CiteSentry  
   • Monitors reference patterns across large biblio-data to catch irregular or inconsistent citations in academic papers.  

   Data Sources (3+):  
   • CrossRef (api.crossref.org)  
   • PubMed (api.ncbi.nlm.nih.gov)  
   • arXiv (export.arxiv.org/api)  

   Async Python Implementation (25 lines):
   ─────────────────────────────────────────────────────────────────────
   import asyncio
   import aiohttp

   CROSSREF_URL = "https://api.crossref.org/works?rows=5&query=climate"
   PUBMED_URL = "https://api.ncbi.nlm.nih.gov/lit/ctxp/v1/pubmed/?format=json&id=30049270"
   ARXIV_URL = "http://export.arxiv.org/api/query?search_query=all:climate&start=0&max_results=5"

   async def fetch_data(session, url):
       async with session.get(url) as resp:
           return await resp.text()  # Some APIs return XML or text

   async def check_citation_integrity():
       async with aiohttp.ClientSession() as session:
           crossref_data = await fetch_data(session, CROSSREF_URL)
           pubmed_data = await fetch_data(session, PUBMED_URL)
           arxiv_data = await fetch_data(session, ARXIV_URL)
           # Simplistic parsing and cross-checking placeholder
           suspicious = []
           if "retracted" in pubmed_data.lower():
               suspicious.append("Potential retracted paper found in PubMed references.")
           return suspicious

   def main():
       results = asyncio.run(check_citation_integrity())
       print(results)
   ─────────────────────────────────────────────────────────────────────
   What It Detects: Repeated or missing citations, contradictory retraction statuses, and suspicious references that slip past standard editorial checks.
