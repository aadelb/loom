# Loom MCP Tools Reference

Complete reference for all tools registered in the Loom MCP research server.

## Search and Discovery Tools

### research_search
Web search via Exa, Tavily, Firecrawl, or Brave with provider cascade fallback.

| Parameter | Type | Default | Description |
|---|---|---|---|
| `query` | string | *required* | Search query |
| `provider` | string | `"exa"` | Preferred provider: `exa`, `tavily`, `firecrawl`, `brave` |
| `n` | int | `10` | Number of results |
| `include_domains` | array | `null` | Limit search to these domains |
| `exclude_domains` | array | `null` | Exclude these domains |
| `start_date` | string | `null` | ISO date (yyyy-mm-dd) |
| `end_date` | string | `null` | ISO date (yyyy-mm-dd) |
| `language` | string | `null` | Language hint (e.g., "en") |
| `provider_config`| dict | `null` | Pass-through config (e.g., `{"semantic": true}`) |

**Required API Key**: `EXA_API_KEY`, `TAVILY_API_KEY`, `FIRECRAWL_API_KEY`, or `BRAVE_API_KEY`

**Example Usage**:
```python
await session.call_tool("research_search", {"query": "AI safety 2026", "provider": "exa"})
```

**Return Shape**:
```json
{
  "provider": "exa",
  "results": [
    {"title": "Example", "url": "https://example.com", "score": 0.95}
  ]
}
```

# API Keys and Environment Variables

Loom operates using a provider cascade for its tools. You do not need to configure all of these keys. Core fetching, stealth browsers, and scraping tools are completely free. You only need API keys if you want to use the search capabilities or the LLM intelligence pipelines.

## Search Providers

Loom cascades through search providers automatically. Supplying at least one is recommended for search capabilities.

| Environment Variable | Provider | Signup URL | Notes |
|---|---|---|---|
| `EXA_API_KEY` | Exa.ai | [https://exa.ai/](https://exa.ai/) | High-quality semantic search (Recommended default) |
| `TAVILY_API_KEY` | Tavily | [https://tavily.com/](https://tavily.com/) | Built specifically for AI agents |
| `FIRECRAWL_API_KEY` | Firecrawl | [https://firecrawl.dev/](https://firecrawl.dev/) | Advanced deep scraping and search |
| `BRAVE_API_KEY` | Brave Search | [https://brave.com/search/api/](https://brave.com/search/api/) | Offers a free tier limit |

## LLM Providers

LLM providers power the intelligence tasks (`research_llm_*` tools, summaries, data extraction, and creative pipelines). Loom cascades failures automatically.

| Environment Variable | Provider | Signup URL | Notes |
|---|---|---|---|
| `NVIDIA_API_KEY` | NVIDIA NIM | [https://build.nvidia.com/](https://build.nvidia.com/) | Features a generous free tier for top open models |
| `OPENAI_API_KEY` | OpenAI | [https://platform.openai.com/](https://platform.openai.com/) | Access to GPT-4o and GPT-4o-mini |
| `ANTHROPIC_API_KEY` | Anthropic | [https://console.anthropic.com/](https://console.anthropic.com/) | Access to the Claude 3.5 model family |
| `VLLM_API_BASE` | vLLM (Local) | *Self-Hosted* | Base URL for self-hosted open-source models |

## General Integrations

| Environment Variable | Provider | Signup URL | Notes |
|---|---|---|---|
| `GITHUB_TOKEN` | GitHub | [https://github.com/settings/tokens](https://github.com/settings/tokens) | Optional. Unlocks higher rate limits for `research_github` commands |
